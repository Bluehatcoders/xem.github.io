Text alignment

june/july 2013

I developed a pure CSS solution that allows to center some text when it's short, and make it align to the left when it's long. (IE6+)

Then, I made a more generic solution that allows to center the text in a defined zone when it's short, and enlarging this zone when it's long (IE8+).

Simple case: text "center-left"

HTML code:

<div class=align-outer>
	<div class=align-inner>
		Text text text
	</div>
</div>

CSS code:

.align-outer {
	text-align: center;
}
.align-inner {
	margin: 0 auto;
	text-align: left;
	display: inline-block;
	*display: inline; *zoom: 1; /* IE < 8 */
}

Demo:
(the wrappers around the text are dotted in red, so you can see them stretching)

Text text text text text




More complex case: text "centered-stretchable-left"

HTML code:

<div class=align-stretch-outer>
	<div class=align-stretch-inner-1>
		<div class=align-stretch-inner-2>
			Texte texte
		</div>
	</div>
</div>

CSS code:

.align-stretch-outer {
	text-align: left;
}
.align-stretch-inner-1 {
  display: inline-block;
  text-align: center;
  min-width: 60%;
}
.align-stretch-inner-2 {
  display: inline-block;
  text-align: left;
}

Demo:

NB: of course, you can replace left by "right" or "justified" in the first CSS rule.

Lightml

march 2013

HTML pages contain a lot of redundancy. This experiment aims to compress HTML server-side (in PHP), and uncompress it client-side (in JavaScript). Results show that we can save up to 46% bandwith (without Gzip), and up to 10% with Gzip enabled.

https://github.com/xem/lightml
